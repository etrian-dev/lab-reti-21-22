The Project

A simple filserver entirely based in main memory (no permanent storage) implemented for the Operating Systems course at Unipi. Clients can connect to the server to upload files and retrieve them. The server has a fixed maximum capacity, and implements a FIFO queue to send files back to clients when a new file won't fit.
